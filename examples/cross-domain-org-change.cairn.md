# Cross-Domain Example: Organisational Change with Psychological and Sociological Elements

## CONTEXT

- Org undergoing transformation.
- Involves individual emotions (psych), power dynamics (org), group norms (socio).

## REQUIREMENTS

R1. Change must address emotional resistance. [MUST]
R2. Coalition building required. [MUST]
R3. Norms updated. [MUST]

## OUTCOMES

Successful adoption with new culture.

EMERGENT [TYPE: organisational; FROM: feedback loops]

```
PROCESS OrgChangeWithHumanElements (INPUT: misalignment; OUTPUT: transformed state)
  STATE
    emotions [scope: individual; dir: read/write]
    power_map [scope: org; dir: read/write]
    norms [scope: group; dir: read/write]
    team_state [scope: org.team; dir: read/write]

  1. APPRAISAL [TYPE: primary] of current state. [EMOTIONAL, STRATEGIC]
  2. COALITION [BUILD] Build guiding coalition. [POWER, LEADERSHIP]
  3. REGULATION [STRATEGY: reappraisal] Regulate resistance. [REGULATION]
  4. SOCIALIZE [TYPE: primary] Socialize new norms. [SOCIAL, GROUP]
  5. FEEDBACK [FROM: emotions; VIA: group] Feedback to adjust.
  6. ALIGN [ELEMENTS: strategy,culture] Align elements. [ALIGN]
  7. MACRO VISION-CASCADE for communication
  8. STATE UPDATE: team_state ← aligned
  9. EMERGENT [SOCIAL: new equilibrium]

  ITERATE [UNTIL: adoption]
