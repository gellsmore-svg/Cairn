# Psychological: Attachment System Activation and Regulation

Cairn description of attachment processes (Bowlby, Ainsworth, modern extensions) as a regulatory system for felt security in relationships.

## CONTEXT

- **Individual**: With internal working models of self and others.
- **Attachment figure(s)**: Caregivers, partners, etc.
- **Threat**: Separation, danger, or unavailability.
- **Goal**: Felt security and proximity.

## REQUIREMENTS

```
R1. The attachment system activates in response to perceived threat. [MUST]
R2. Behaviors aim to achieve or maintain proximity to attachment figures. [MUST]
R3. Successful regulation updates working models. [MUST]
R4. Insecure patterns can lead to maladaptive strategies. [MUST]
```

## OUTCOMES

Restored sense of security, or escalation if regulation fails.

EMERGENT [PSYCHOLOGICAL: updated attachment style or relational pattern]

## PROCESS — Formal

```
PROCESS AttachmentRegulation (INPUT: perceived threat to bond; OUTPUT: restored security)
  1. APPRAISAL [TYPE: primary] of threat and availability of attachment figure. [APPRAISAL, EMOTIONAL]
  2. Activate attachment system (protest, seek proximity, or deactivate). [BEHAVIORAL, EMOTIONAL]
  3. REGULATION [STRATEGY: co-regulation] via co-regulation (with other) or self-soothing strategies. [REGULATION]
  4. STATE UPDATE: working model refined based on outcome.
  5. EMERGENT [SATISFIES: R3]  # reinforced or revised relational expectations

  ITERATE in ongoing relationships.
```
