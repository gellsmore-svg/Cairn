# Organisational: McKinsey 7-S Alignment Model

Cairn description of the McKinsey 7-S Framework for diagnosing and achieving organisational effectiveness through alignment.

## CONTEXT

- **Hard elements**: Strategy, Structure, Systems.
- **Soft elements**: Shared Values, Skills, Style, Staff.
- **Organisation**: Needs internal consistency for high performance.

## REQUIREMENTS

```
R1. All 7 elements must be aligned for effectiveness. [MUST]
R2. Change in one element requires adjustments in others. [MUST]
R3. Shared values are central and influence the others. [MUST]
R4. Diagnosis must cover both hard and soft factors. [MUST]
```

## OUTCOMES

High organisational effectiveness through coherent configuration of elements.

EMERGENT [ORGANISATIONAL: superior performance and adaptability]

---

## PROCESS — Formal

```
PROCESS Align7S (INPUT: performance issues; OUTPUT: aligned organisation)
  STATE
    strategy      [scope: org; dir: read/write]
    structure     [scope: org; dir: read/write]
    systems       [scope: org; dir: read/write]
    shared_values [scope: org; dir: read/write]
    skills        [scope: org; dir: read/write]
    style         [scope: org; dir: read/write]
    staff         [scope: org; dir: read/write]

  1. Diagnose current state of all 7 elements. [STRATEGIC, ALIGNMENT]
  2. Identify misalignments.
  3. DECISION [ON: which elements to change first]
  4. Design interventions to restore fit across elements. [CULTURAL, STRUCTURAL]
  5. Implement and monitor alignment.
  6. ITERATE as needed.

  # EMERGENT [SATISFIES: R2]  # internal consistency and effectiveness
```
