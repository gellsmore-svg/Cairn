# Organisational: Kotter's 8-Step Change Model

Cairn description of John Kotter's 8-Step Model for Leading Change.

This is a high-level strategic process for large-scale organisational transformation.

## CONTEXT

- **Organisation** facing disruption or opportunity.
- **Leadership** team driving change.
- **Employees** at all levels.
- **Culture** as the deep structure to be shifted.

## REQUIREMENTS

```
R1. Change requires creating a sense of urgency. [MUST]
R2. A powerful guiding coalition is essential. [MUST]
R3. Vision must be clear and compelling. [MUST]
R4. Communication must be frequent and consistent. [MUST]
R5. Obstacles must be removed and short-term wins generated. [MUST]
R6. Changes must be anchored in culture. [MUST]
```

## OUTCOMES

The organisation successfully transforms, with new behaviours embedded and sustained.

EMERGENT [SATISFIES: R6]  # lasting behavioural and structural shift

## PROCESS — Formal

```
PROCESS LeadChange (INPUT: need for transformation; OUTPUT: transformed organisation)
  STATE
    urgency   [scope: org; dir: read/write]
    coalition [scope: org; dir: read/write]
    vision    [scope: org; dir: read/write]
    culture   [scope: org; dir: read/write]

  1. Create a sense of URGENCY. [LEADERSHIP, STRATEGIC]
  2. COALITION BUILD a powerful GUIDING COALITION. [POWER]
  3. VISION Create a VISION to help direct the change. [VISION]
  4. CASCADE COMMUNICATE the vision widely and repeatedly. [CASCADE]
  5. RESISTANCE EMPOWER broad-based action by removing obstacles. [RESISTANCE]
  6. REINFORCEMENT Generate SHORT-TERM WINS. [REINFORCEMENT]
  7. CONSOLIDATE gains and produce more change.
  8. ALIGN ANCHOR new approaches in the culture. [CULTURE, ALIGNMENT]

  # EMERGENT [SATISFIES: R6]  # sustained new performance and behaviours
```

## PROCESS — Operator Profile

```
render-profile: operator

  Owner: Senior leadership + coalition
  Assisted-by: Communication, quick wins, cultural levers

  8. Embed in culture
```
